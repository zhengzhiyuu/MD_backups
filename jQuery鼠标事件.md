- #### {% post_link mousedown与mouseup事件 mousedown与mouseup事件 %}

- #### {% post_link mousemove事件 mousemove事件 %}
mousemove方法可以监听用户鼠标移动的操作
- #### {% post_link mouseover与mouseout事件 mouseover与mouseout事件 %}
